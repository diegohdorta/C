Installation
============

After follow the setup you are now able to successfully install PyIMX library in your computer.

Obtaining the Package
~~~~~~~~~~~~~~~~~~~~~

You can download the latest version by clicking on the proper link below:

+-------------------+----------------+-----------------------------------------------------------------+
| Download Type     | Python Version | Location                                                        |
+===================+================+=================================================================+
| Source tarball    | 3.x or Higher  |  (Not done yet)                                                 |
+-------------------+----------------+-----------------------------------------------------------------+
| Development       | 3.x or Higher  | `PyIMX GitHub Repo <https://github.com/diegohdorta/pyimx>`_     |
+-------------------+----------------+-----------------------------------------------------------------+

Installing
~~~~~~~~~~

To install from source do::

    $ tar -xf pyimx-x.x.x.tar.gz
    $ cd pyimx-x.x.x/
    $ python3.x setup.py build
    $ python3.x setup.py install

To install using pip3::

    $ sudo pip3 install pyimx
    
Getting Help
~~~~~~~~~~~~

For questions, bug reports, feature request, please consider using the following methods:

 1.  Create an Issue for PyIMX on `GitHub <https://github.com/diegohdorta/pyimx/issues>`_.

 2.  If you're sure you've found a bug in existing code, or have some code
     you think would be useful to add to PyIMX create a pull request on GitHub.

 3.  In case you need help to start developing PyIMX, read the section **How to contribute**.
